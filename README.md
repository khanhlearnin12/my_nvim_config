1. install nvim 
```bash
sudo snap install nvim --classic 
```
2. clone the repo
```bash 
git clone https://github.com/khanhlearnin12/my_nvim_config.git
```
3. go to my_nvim_config direc to mv all the file to `.config\nvim`
```bash
cd my_nvim_config 
mv * ~/.config/nvim
```
